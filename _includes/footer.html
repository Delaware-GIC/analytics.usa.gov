About this Site

This data provides a window into how people are interacting with the State of Delaware government online. The data comes from Google Analytics accounts for State of Delaware government agencies. This data helps government agencies understand how people find, access, and use government services online. This does not track individuals, and anonymizes the IP addresses of visitors.

Not every State of Delaware website is represented in this data. We plan to expand the data made available here, and will be adding new domains from agencies and Delaware municipalities that opt-in. To add your site, please contact us.

This website is a fork of an open source project originally released by the Government Service Administration (GSA) in the Federal Government. This program is in the public domain, which means that this website and its data are free for you to use without restriction. You can find the code for this website and the code behind the data collection on GitHub.
